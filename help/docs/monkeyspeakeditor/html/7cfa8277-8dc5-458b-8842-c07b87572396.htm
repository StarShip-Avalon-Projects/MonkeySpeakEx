HelpLink.ini

[This is preliminary documentation and is subject to change.]

HelpLink.ini is a configuration file for the Reference Links dropdown menu component. This file is broken down into a section named after the application product name or an arbitrary name for a different menu. The section keys define the menu option. the KeyName spcefies the text of the menu option. The Key value determins the web link it points to.
