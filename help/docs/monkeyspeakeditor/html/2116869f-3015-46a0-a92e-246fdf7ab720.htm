MonkeySpeak Variables

  1. %MESSAGE = what a furre said in Say, Shout, Emit, emote, whispers or what ever text the server responds with.

  2. %NAME = Triggering Furre's Name.

  3. %MSPATH = The file path to the current Silver Monkey configuration file "*.bini"

    Defaults to "Documents\Silver Monkey".

  4. %BOTNAME = Bots character name on furcadia.

  5. %BANISHNAME = The last name the detected using a furcadia banish command.

  6. %BANISHLIST = The list of names the bot sees on the banish list. The Silver Monkey mirrors the furcadia banish system and tracks changes to the list. This helps reduce the number of calls to the server. The list can be refreshed any time by asking the server for a "banish-list".

  7. %BOTCONTROLLER = The assigned name for Silver Monkey's bot-controller,

  8. %DREAMNAME = Name of the Dream the Bot has joined.

  9. %DREAMOWNER = The Owner of the Dream the Bots in.
